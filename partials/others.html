Vagrant Lab (Virtualbox Lab 2.0)

Vagrant is an absolutely spectacular tool. It works in conjunction with Virtualbox and helps me spin up multiple fully configured virtual machines in minutes. With that I'm able to test out new infrastructure designs, new software, and continually improve on automation.

Right now I'm using it to simulate a production Chef and Docker lab. Each virtual machine will be thought of as a bare metal system. One system serves as the Chef server and manages the other virtual machines on the OS level. The remaining virtual machines will act as Chef clients with Docker running in swarm configuration.
