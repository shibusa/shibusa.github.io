networkConfig

Best viewed in fullscreen.

Vendor agnostic network device configurator written in Python.

networkConfig is designed to minimize basic configuration time of network devices. End users are provided the flexibility to configure the tool to allow for network device vendors of all sorts (Cisco, Juniper, etc.).

multihost

Best viewed in fullscreen.

BASH query tool written in Python.

multihost is designed to do quick queries across large infrastructures. The tool is used to audit configuration inconsisties across 1000+ UNIX systems in pass. This is the inspiration for the Bastion project.
